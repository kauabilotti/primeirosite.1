Explicação de TAG

  • HTML:Abreviação de "HyperText Markup Language", é um componente fundamental da linguagem de marcação HTML usada para criar páginas web.
  • HEAD:Este elemento é usado para conter informações sobre o documento HTML, como o título da página
  • BODY:Contém o conteúdo principal do documento HTML, como texto, imagens, vídeos, etc. Tudo o que é exibido na página web é colocado dentro do elemento
  • HEADER:É usado para definir o cabeçalho de uma página ou de uma seção específica dentro de uma página,ele contém logotipos, títulos, menus de navegação e outros elementos relacionados ao cabeçalho da página.
  • SECTION:Dividir o conteúdo de uma página em seções temáticas ou grupos de conteúdo relacionado.
  • DIV:É um contêiner que agrupa elementos. Ele é frequentemente usado para aplicar estilos CSS ou manipular o conteúdo com JavaScript.
  • FOOTER:Usado para definir o rodapé de uma página ou de uma seção específica dentro de uma página. Geralmente, ele contém informações de contato, links para políticas de privacidade, direitos autorais e outros elementos relacionados ao rodapé da página.
  • H1H6:Esses elementos são usados para definir os cabeçalhos de uma página ou de uma seção específica dentro de uma página. H1 é o cabeçalho mais importante e H6 é o menos importante.Eles são usados para organizar o conteúdo e proporcionar uma estrutura hierárquica à página.
  • HR:Este elemento é usado para criar uma linha horizontal no documento HTML. É frequentemente usado para separar seções de conteúdo.
  • P:É usado para definir um parágrafo de texto.
  • UL:Este elemento é usado para criar uma lista não ordenada de itens. Cada item da lista é definido pelo elemento "li"
  • LI:É usado para definir um item em uma lista
  • BR:Este elemento é usado para inserir uma quebra de linha dentro de um parágrafo ou em outro contexto onde a quebra de linha é necessária.

Mais 10 TAG que Você tem que Conhecer

  • A:Usada para criar hiperlinks, permitindo que os usuários naveguem de uma página para outra ou dentro da mesma página.
  • IMG:Utilizada para inserir imagens em uma página web.
  • STRONG:Serve para enfatizar o texto, geralmente exibindo-o em negrito.
  • IMFRAME:É usado para incorporar uma página web dentro de outra página.
  • TABLE: Muito usada para criar tabelas que organizam e exibem dados em linhas e colunas.
  • OL: Cria listas ordenadas, onde os itens são numerados.
  • CODE:Utilizada para marcar trechos de código de programação em um texto.
  • ABBR: Serve para definir uma abreviação ou um acrônimo, fornecendo uma explicação ou significado completo quando o cursor é colocado sobre o texto.
  • CITE: Usado para indicar o título de uma obra, como um livro, filme ou artigo, dentro de um conteúdo citado.
  • FORM: Utilizada para criar formulários interativos que permitem aos usuários inserir e enviar dados.
